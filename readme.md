

### vue component library boilerplate

**Feature**

- pug loader
- vue
- sass loader
- dev server
- typescript type definition
- tailwind

**更新方法**

- yarn build ( rebuild 專案 type definitions )
- git commit
- git new tag
- git push

**如需要於組件庫內加入其他套件，需考慮 peer dependency （於 package.json 註明)**

第三方引用時，tag 須為最新版本
